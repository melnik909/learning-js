Объект

Свойства упорядочены особым образом: свойства с целочисленными ключами сортируются
по возрастанию, остальные располагаются в порядке создания

«целочисленное свойство» означает строку, которая может быть преобразована в целое
число и обратно без изменений.

То есть, "49" – это целочисленное имя свойства, потому что если его преобразовать в
целое число, а затем обратно в строку, то оно не изменится. А вот свойства "+49" или "1.2"
таковыми не являются:

let codes = {
  "49": "Германия",
  "41": "Швейцария",
  "44": "Великобритания",
  // ..,
  "1": "США"
};

for (let code in codes) {
  console.log(code); // 1, 41, 44, 49
}

Сравнение объектов

Два объекта равны только в том случае, если это один и тот же объект.

let a = {};
let b = a; // копирование по ссылке

alert( a == b ); // true, обе переменные ссылаются на один и тот же объект
alert( a === b ); // true

let a = {};
let b = {}; // два независимых объекта

alert( a == b ); // false
